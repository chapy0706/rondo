# issue-14: 結果発表画面

## 背景

1ゲームの締めとなる結果発表画面。順位や軌跡を見せ、ルームの解散へと繋ぐ。rondo では1ゲームの終了でルームが解散するため、ここがプレイの終点になる。

## スコープ

apps/web に、結果発表画面を実装する。

### このissueでやること

- 結果UI（順位、軌跡）
- ルーム解散への流れ

### このissueでやらないこと

- ランキングの永続化（Phase 2）

## 設計方針

- ルームは1ゲームの終了で解散する（ADR 0017）
- 結果は揮発でよく、永続化しない（ADR 0019）

## 受け入れ条件

- ゲーム終了後に順位が表示される
- 軌跡など、その回の結果が見られる
- 結果のあとルームが解散する

## 段階

1. 結果UIを実装する
2. 順位・軌跡を表示する
3. ルーム解散への流れを繋ぐ
