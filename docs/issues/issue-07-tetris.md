# issue-07: ソロゲーム テトリス

## 背景

テトリスは Phase 1 のソロゲーム第1号であり、「プラグインとしてゲームを足せる」ことの実証を担う。基盤との接点がマニフェストとスコア返却だけで済むため、メンバーが担当しやすく、Gleam に触れる必要もない。

## スコープ

apps/web/src/games/tetris に、ソロゲームとしてのテトリスを実装する。

### このissueでやること

- テトリス本体のロジックと描画
- マニフェストの作成と registry への登録
- VirtualPad による入力
- スコアの基盤への返却

### このissueでやらないこと

- リアルタイム要素・通信

## 設計方針

- SoloGame として実装し、リアルタイム接続を受け取らない（ADR 0004）
- 入力は基盤共通の VirtualPad を使う（ADR 0018）
- マニフェストで自身を記述し、registry に登録する（ADR 0003）

## 受け入れ条件

- ゲーム選択画面から起動して遊べる
- VirtualPad で操作できる
- 終了時にスコアが基盤に返る
- 追加にあたり基盤側のコードを変更していない

## 段階

1. マニフェストを作り registry に登録する
2. テトリス本体を実装する
3. VirtualPad 入力を繋ぐ
4. スコア返却を繋ぐ
