# ADR 0015: プレイヤー識別は端末保持の表示名とし、永続アカウントを持たない

- ステータス: 受理
- 日付: 2026-06-24

## 文脈

プレイヤーをどう識別するかを決める必要がある。アカウント登録を求めると気軽さが削がれる一方、誰が誰かを区別する最低限の手段は要る。

## 決定

Phase 1 では永続アカウントを持たない。プレイヤーはデフォルトで user1, user2, ... と割り当てられ、設定画面で表示名を変更できる。表示名は端末に保持する。

## 理由

- 「ちょっと集まって、また明日」の軽さに、アカウントの壁は合わない
- デフォルト名を即座に与えることで、何も入力せずに遊び始められる
- 永続的な識別が必要になるランキング等は Phase 2 の課題とし、必要になってから足す

## 検討した代替案

- 登録制アカウントを Phase 1 から導入する案。気軽さを損ない、スコープを膨らませるため却下した
- 完全匿名（名前なし）の案。プレイヤーの区別がつかず、対戦の体験が薄くなるため却下した

## 結果

- 初回はデフォルト名を自動付与する
- 表示名の変更と端末保持を Phase 1 の範囲に含める（ADR 0019 と整合）
