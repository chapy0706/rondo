# ADR 0010: 新規リポジトリとして作る

- ステータス: 受理
- 日付: 2026-06-24

## 文脈

かくれんぼ単体のリポジトリが既に存在する。これをプラットフォーム化するにあたり、既存リポジトリを改修するか、新規に作り直すかを決める必要がある。

## 決定

新規リポジトリ rondo として作り直す。かくれんぼリポジトリのコードは引き継がず、設計資産としてのみ参照する。

## 理由

- 既存リポジトリは「かくれんぼ」という1ゲーム名で切られており、プラットフォームの器として名前から合わない
- コンセプトが1ゲームからプラットフォームへ変わるため、git 履歴的にも新規が素直
- 過去の README の構造は良くできているので、設計の参考として残す

## 検討した代替案

- 既存リポジトリを改修して流用する案。名前とコンセプトの不一致が残り、後から見て混乱を生むため却下した

## 結果

- rondo を新規リポジトリとして作成する
- かくれんぼの README とディレクトリ設計は参考資産として保持する
